My work focuses on how AI can help people preserve context, make better decisions, define operational loops, review messy conversations, and continue complex projects across tools and sessions.

Website: [aizhiz.com](https://aizhiz.com)

## Main Direction

I am currently building around three connected layers:

1. **FlowGrid** — project memory and continuity protocol for AI-assisted knowledge work
2. **FDE Operator OS** — applied AI delivery playbook for turning messy business reality into operator loops
3. **Biz Retro Analyzer** — evidence-first dialogue intelligence for complex project conversations

Together, these projects explore one practical question:

> How can AI become part of real work systems, where context, judgment, decisions, and delivery paths are preserved across time?

## 1. FlowGrid

FlowGrid is my flagship project.

It is a local-first project memory protocol for AI-assisted knowledge work.

FlowGrid helps strategy, marketing, research, operations, and solution professionals turn vague business work into structured, auditable, and transferable project systems.

Core ideas:

- project truth lives in local files
- decisions are recorded with rationale and reversal conditions
- AI agents can resume from the same project state
- medium/high-risk changes stay reviewable before merge
- project context becomes portable across tools and sessions

Repository: [`dlxeva/FlowGrid`](https://github.com/dlxeva/FlowGrid)

## 2. FDE Operator OS

FDE Operator OS is an applied AI delivery playbook for forward-deployed and field-facing AI work.

It helps turn messy customer situations into delivery-worthy AI operating loops.

The repository focuses on:

- qualifying whether an AI opportunity is worth pursuing
- modeling operational reality
- finding the real system bottleneck
- defining operators, triggers, business objects, states, actions, and evidence
- compressing broad ideas into minimum viable loops
- designing pilot boundaries and acceptance contracts

Repository: [`dlxeva/fde-operator-os`](https://github.com/dlxeva/fde-operator-os)

## 3. Biz Retro Analyzer

Biz Retro Analyzer is an evidence-first dialogue intelligence skill for messy project conversations.

It turns meeting notes, transcripts, field observations, and follow-up reasoning into structured facts, claims, influence chains, judgment audits, and next actions.

It is designed for situations where ordinary summaries are too smooth and too shallow.

Repository: [`dlxeva/biz-retro-analyzer`](https://github.com/dlxeva/biz-retro-analyzer)

- Separate facts, claims, decisions, and assumptions
